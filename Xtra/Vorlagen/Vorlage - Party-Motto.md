# Checkliste

## Schatzsuche

- [ ] Rahmengeschichte für die Schatzsuche erstellen
- [ ] 4 bis 5 Rätsel aussuchen, die zur Geschichte passen
- [ ] Übersicht über die Rätsel und das Material erstellen
- [ ] Grafiken für Charaktere und Hintergründe suchen
- [ ] Materialien erstellen
- [ ] Icons für die einzelnen Rätsel erstellen und in Materialien einfügen
- [ ] Markierungskarten für die Stationen erstellen
- [ ] Wegweiser-Karten erstellen
- [ ] Schatzsuche testen
- [ ] Nach dem Test notwendige Anpassungen an der Schatzsuche machen
- [ ] Druckvorlage erstellen
- [ ] Anleitung erstellen
- [ ] Materialien in die Dropbox laden und Download-Link auf Website erstellen
- [ ] Shop-Grafiken erstellen
- [ ] Download-PDF erstellen
- [ ] Etsy-Listing erstellen

## Einladungskarte

- [ ] Einladungskarte erstellen
- [ ] Canva-Vorlage erstellen
- [ ] Download-Paket erstellen und in die Dropbox laden
- [ ] Shopgrafiken erstellen
- [ ] Einladungskarte im Shop einstellen
- [ ] Pins für Einladung erstellen

## Zubehör

- [ ] Cupcake-Topper erstellen
- [ ] Etikett für Mitgebsel-Tüte erstellen
- [ ] Download-Paket für das Zubehör erstellen und in die Dropbox laden
- [ ] Shopgrafiken für das Party-Zubehör erstellen
- [ ] Zubehör im Shop einstellen
- [ ] Pins für das Zubehör erstellen und einplanen

## Sonstige

- [ ] Party-Bundle erstellen
- [ ] Schatzsuchen-Bundle erstellen
- [ ] Blogartikel erstellen
- [ ] Pins für Bundles erstellen
